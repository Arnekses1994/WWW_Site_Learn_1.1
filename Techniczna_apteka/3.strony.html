Przydatne strony www którch warto używac w pracy przed pierwszym stołem, dobrym sposobem na łatwiejsze ich używanie jest przypiecie ich na stałe do przeglądrki oraz właczanie codzinnie rano w pracy :)

1.www.Ktomalek.pl ( Szukanie leków )

Dla wielu pewnie znana strona lecz to podstawa o której musze wspomnieć.Bardo prosta w obsłuche, pokazuje nam po wpisaniu miejscowośći oraz nazwy leku wraz z dawką najbliższa lokalizacje medykamnetu co napewno ułatwia rozmowę z pacjentem jak poszukiwania.Dodatkowo zawieraja sie tam informacja o kontakcie z apteka ( Telefon ) oraz godzinach otwarcia więć pacjent wie czy mając mała ilość czasu zdąży wukpic lek

2.www.Gziepolek.pl ( Szukanie leków )

Bardzo podobna strona jak wyżej wspomniana jedynie tutaj baza aptek ( apteka sama musi sie do takowej dopisać ) jest mniejsza co za tym idzie nie tak wygodne jak serwis ktomalek.pl lecz mimo to jak nie widzimy w okolicy warto sprawdzic też na tej stronie.Dla nas to poświecona chwila a dla pacjenta duże ułatwienie

3.www.Gov.pl/web/gis/ostrzezenia ( GIS )

Portal Głównego Inspektora Sanitarnego a dokłądniej lista ostrzeżeń aktualizowanych przez GIS.Warto raz ja jakis czas tam spojrzeć Zdarzają sie preparaty apteczne które apteka musi kontrolować.Najlepiej wyrobic sobie nawyk i codziennie rano sprawdzać liste. Przydatne też jest drukowanie takuch decyzji ( Programem do robienie zrzutów ekranu ) i którka notka z strony personelu,że zostało wszystko sprwadzone.Podczas kontroli nieoceniona pomoc - Sprawdzone na własnej skórze

4.www.Rdg.ezdrowie.gov.pl ( GIF )

Portal Głośnego Inspektora Farmaceutycznego a dokładniej lista decyzji wydanych przez wspomniany organ.Turaj radze juz codziennie sprawdzać decyzje i każda drukować wraz z którka notka o sprawdzonych leku + pieczątka.Podczas Kontroli takżę bardzo ważna rzecz ale ułątwi sprawdzalność takich decyzje a czasem potrafi być ich kilka dzinnie.

5.www.Zatrzymajpacjenta.pl/baza-interakcji ( Interakcje leków )

Wiekszość aptek ma taki modól juz ustawiony w kamsofcie ( podczas podliczania recepty ) lecz gdy danego leku nie posiadanmy to nie można szybko sprawdzic interakcji a przy kilku lekach może byc to dla nas wyzwaniem.Dzięki tej stronie nawet w domowym zaciszu sprawdzimy wiele medykamentów.

6.www.Bartoszmowi.pl ( Prosty Indeks Leków )

Bardzo prosty indeks leków przydatny do szybkie podejrzenia działania leku, odpłatnośći bez zbędnych informacji.Przydatny gdy nie możemy sobie przypomnieć działania rzadko spotykanego leku lub na szybko chcemy podejrzec zapłate.Jest aktualizowane nabierząco więc nie ba obaw o stare dane ceny leku

7.www.Zamienniki.pharmindex.pl ( Sprawdzanie podziału leków i temperatur )

Podstrona serwisu Pharmindex tutaj darmowa bardzo wygodna do sprawdzania podzielnośći tabletki w opakowaniu oraz szybkiego podglądu na temperature przechowywania leku.Dodatkowo dorzuca informacje o wypłeniająych składniach ( np.laktoza ) oraz dostępnośći leku.
